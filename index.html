Zero to SOLID in 45 Minutes

This is sample source for the Zero to SOLID in 45 Minutes talk for the GTA-PHP Meetup Group. It is a mock implementation of a simple shopping cart. Slides are available at Slideshare.

The code is split into branches for each stage of evolution covered in the talk. It progresses from "Zero" which is a sample of what beginner PHP might look like to SOLID, which is object oriented code that - at least in some way, follows the principals of SOLID. These branches are numbered as follows.

  • 00: "Zero" - no classes, not even a function defined.
  • 01: Code moved into functions
  • 02: Functions moved into a "god" object
  • 03: SRP - class split into responsibilities
  • 04: OCP - classes modified to encourage extension without modification
  • 05: LSP - gave up on a code example. Too contrived. No such branch.
  • 06: DIP - dependance on concrete classes changed to interfaces
  • 07: ISP - interface(s) divided to be client specific
